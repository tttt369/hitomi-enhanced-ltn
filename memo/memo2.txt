.index: keyに対応する.dataの範囲を記したファイル
.data: 実際のデータ

1.
検索ワードを
const key = new Uint8Array(sha256.array(query).slice(0, 4))
このようにsha256にし、4バイト事に区切ったリストにする

2.
日付ごとに変わるindexファイルにaddress; 0からstep; 463の範囲のヘッダーでバイトを取り出す

3.
キー、データ、アドレスに別れたデータセクションがあり、そのうち最初の２つは、先頭はデータ個数、その後個数分のデータある。また、それぞれのデータの間には４バイト間が空いているため。
データ読み取り -> pos += 4 -> indexからのバイト.slice(pos, pos + 読み取ったデータのバイト数) -> キー
このように足しながら移動
最後の一つは
const B = 16;
const number_of_subnode_addresses = B + 1;
このようにハードコードされた回数分データを読み取る

4.
先ほど取り出したキーリストの要素と検索ワードのキーを比較する
要素、キーの４バイトを１つの塊として比較、データリスト[一致したキーリストのインデックス]を返す
一致しなかった場合、アドレスリスト[キーリストの個数]この値を2のindexファイルのaddressにして。新たにデータを取り出し
キーの一致でfalseが出るまでこれを繰り返す

5.
4で帰ってきたデータから2のインデックスファイルのaddress; データ[0], step; データ[1]このように割り当てる。また.dataを拡張子として取り出す。

6.
5で取り出したデータは３の最初の２つのような構造をしていて、最初にidの個数その次に個数分のデータがある。データ間のバイトも同じく４バイト、ループ処理でデータをリストに追加してしまえば完成
